# Sodium Relief

Sodium Relief is a lightweight client-side companion for Sodium focused on one problem: reducing tooltip and hover stutter on inventory-like screens.

It is designed for GUI interactions where repeated tooltip work can feel uneven even when overall FPS is already fine.

## Features

- smoother tooltip behavior on repeated hover
- less redundant tooltip recalculation on supported screens
- steadier inventory, chest, and similar GUI interactions
- optional in-game configuration through Sodium or Mod Menu

## Scope

Sodium Relief is intentionally narrow in scope.

It does not try to replace Sodium or act as a general FPS mod. Its purpose is to improve the feel of tooltip-heavy GUI usage.

## Compatibility

- Minecraft 1.21.11
- Fabric Loader
- Fabric API
- Java 21
- client-side only

## Installation

Place Sodium Relief in your `mods` folder.

Sodium is recommended. Mod Menu is optional.
